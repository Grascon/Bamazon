Bamazon Customer

When running node bamazonCustomer.js you will be displayed Bamazon's awesome inventory stored in a MySQL Database. When displaying Bamazon's inventory you will be able to see the Product's I.D, Name, and Price. You will then be asked to enter the I.D of the product you wish to purchase and how much you would like to purchase. When completing the order you will be given the total price and an order summary.

Bamazon Manager

When running node bamazonManager.js you will be displayed Bamazon's Manager Menu and be able to select from the options seen below:

Selecting "View Products for Sale" allows you to view the entire inventory stored in a MySQL Database.

Selecting "View Low Inventory" allows you to view the products that have less than 5 in stock.

Selecting "Add to Inventory" allows you to add to the products stock and updates the database in MySQL.

Selecting "Exit" with have you exit from the menu.
